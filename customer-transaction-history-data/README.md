# Customer Transaction History Data

This exercise demonstrates summarizing data of the transaction history of several customers.

## Key Details

1. Orders Sorted by Year (descending)
2. Total Revenue per Category
3. Top 2 Customers by Total Spending
4. Categories Where Total Revenue is Greater Than $1000
5. Average Order Value per Year
6. Row Count
7. Customer Count
8. Max Price
9. Min Price
10. Average Price
